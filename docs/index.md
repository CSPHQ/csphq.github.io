Collect, Save, Produce

我们收集，我们保存，我们生产

针对一切关于人工智能的内容

- Collect，收集互联网上关于人工智能的信息，符合作者的许可协议的前提下，扩充知识；
- Save，保存这些知识，保护这些知识，必要时将他们转换为更容易保存与使用的格式；
- Produce，生产新的知识，提供更方面的方法，更合理的工具，更容易的知识检索方式。
